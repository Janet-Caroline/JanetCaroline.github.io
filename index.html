Latest Project

HEB PILOT

HEB Pilot involves automating the process of Requesting for a new authorized software. This request is raised by the Heb user using salesforce App cloud to get a new desktop software installed in their machine. Serial Number for software installation are carried out by NOC team members of HEB. A new purchase order is created for each purchase request and the purchase order is downloaded from the app cloud and sent to the vendor for invoice processing. The details in the invoice are then extracted and fed into the app cloud for approval. the installation of the software takes place manually once the purchase is approved.
